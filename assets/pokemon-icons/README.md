# Pokemon Icon Catalog

This folder keeps the Pokemon icon manifest contract and local cache policy.

Pokemon Showdown remains the canonical battle and species data source. PokeAPI sprites are approved only as visual enrichment assets for local cached report icons.

Committed files:

- `manifest.json` is the schema/skeleton contract.
- `README.md` documents source and storage policy.

Ignored local files:

- `manifest.local.json` is generated by the sync helper.
- `icons/` contains downloaded Pokemon icon PNG files.

Downloaded Pokemon icon files must not be committed. The sync helper should use PokeAPI sprite URLs, write local cache files under `icons/`, and update `manifest.local.json`. Paper Report rendering must use local cached icons only; it must never fetch icons from the network while generating a report. Missing icons must safely fall back to text/name-only output.
